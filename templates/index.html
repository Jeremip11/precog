Jekit

Preview your Github-hosted websites before making them live. Use it to check your plain static or Jekyll-generated websites before you make them live to Github Pages or to your own server.

Check it, before you wreck it.

Preview pages in Jekit by adding your Github account name and repository name to a Jekit URL:

{{request.host}}/<account>/<repo>/

Example showing the Beyond Transparency book site:
{{request.host}}/codeforamerica/beyondtransparency/

To view a commit or branch other than master, add the ref to the end:

{{request.host}}/<account>/<repo>/<ref>/

Example showing a single Beyond Transparency commit:
{{request.host}}/codeforamerica/beyondtransparency/08463604/

To view a specific file or directory, add it after the ref:

{{request.host}}/<account>/<repo>/<ref>/wow/such-path.html

Example of the Beyond Transparency preface:
{{request.host}}/codeforamerica/beyondtransparency/master/chapters/preface/
